## 2. Реализованные типы, протоколы и контейнеры

### 2.1. TypeVar (обобщённые типы)

| TypeVar | Описание |
|---------|----------|
| `T` | Общий тип для элементов коллекции |
| `R` | Тип результата при преобразовании (map) |
| `D` | TypeVar с bound на протокол Displayable |
| `S` | TypeVar с bound на протокол Scorable |

### 2.2. Protocols (протоколы)

| Протокол | Метод | Описание |
|----------|-------|----------|
| `Displayable` | `display() -> str` | Возвращает краткую информацию об элементе |
| `Scorable` | `score() -> float` | Возвращает стоимость или оценку элемента |

### 2.3. TypedCollection (Generic контейнер)

| Метод | Сигнатура | Описание |
|-------|-----------|----------|
| `find` | `(predicate: Callable[[T], bool]) -> Optional[T]` | Поиск первого элемента по условию |
| `filter` | `(predicate: Callable[[T], bool]) -> List[T]` | Фильтрация элементов |
| `map` | `(transform: Callable[[T], R]) -> List[R]` | Преобразование типов |
| `__len__` | `() -> int` | Количество элементов |
| `__getitem__` | `(index: int) -> T` | Доступ по индексу |

### 2.4. Вспомогательные функции

| Функция | Сигнатура | Описание |
|---------|-----------|----------|
| `display_all` | `(items: TypedCollection[D]) -> List[str]` | Отображение всех элементов |
| `total_score` | `(items: TypedCollection[S]) -> float` | Суммарная стоимость |
