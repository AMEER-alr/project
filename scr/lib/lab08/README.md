Изучение основ объектно-ориентированного программирования в Python, создание модели данных с помощью @dataclass, реализация методов и сериализация объектов.

## Структура класса Student

### Поля класса:
- `fio` (str): ФИО студента
- `birthdate` (str): Дата рождения в формате YYYY-MM-DD
- `group` (str): Группа (например, SE-01)
- `gpa` (float): Средний балл (от 0 до 5)

### Методы класса:
1. `__post_init__()`: Валидация данных при создании объекта
2. `age()`: Вычисление возраста студента
3. `to_dict()`: Сериализация в словарь
4. `from_dict()`: Статический метод для десериализации
5. `__str__()`: Строковое представление объекта

## Примеры использования

### Создание объектов Student
```python
from models import Student

student1 = Student(
    fio="Иванов Иван Иванович",
    birthdate="2000-05-15",
    group="SE-01",
    gpa=4.5
)

print(student1)

print(f"Возраст: {student1.age()} лет")
print(f"Данные в виде словаря: {student1.to_dict()}")
